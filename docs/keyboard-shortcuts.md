# Keyboard Shortcuts

Agile Management Portal supports keyboard shortcuts for fast navigation. On macOS use the **Cmd** key; on Windows and Linux use **Ctrl**.

## Navigation

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + 1` | Go to Dashboard |
| `Cmd/Ctrl + 2` | Go to Organization |
| `Cmd/Ctrl + 3` | Go to People |
| `Cmd/Ctrl + I` | Go to Import |

## Search

| Shortcut | Action |
|----------|--------|
| `Cmd/Ctrl + K` | Open global search |
| `Escape` | Close search overlay |

## Modals

| Shortcut | Action |
|----------|--------|
| `Escape` | Close the active modal dialog |

## Notes

- Shortcuts are disabled when a text input, textarea, or dropdown is focused, to avoid interfering with typing. The **Escape** key is an exception — it always works for closing search and modals.
- All shortcuts use the standard modifier key for your platform: **Cmd** on macOS, **Ctrl** on Windows/Linux.
