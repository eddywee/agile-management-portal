# Agile Management Portal

**Agile Management Portal** is a cross-platform desktop application for tracking FTE (Full-Time Equivalent) allocation across SAFe (Scaled Agile Framework) organizational hierarchies.

Built for portfolio managers, RTEs, and agile leaders who need to plan and visualize how people are allocated across Solutions, Agile Release Trains (ARTs), and Product Teams — per Program Increment (PI).

## Key Features

- **Dashboard** — KPI cards, FTE bar charts, role distribution donut chart, and an over-allocation conflict table
- **Organization Tree** — Hierarchical view of Solutions, ARTs, and Teams with leadership and membership management
- **People Management** — Searchable people directory with per-person allocation details and PI history
- **CSV Import Wizard** — Four-step guided import for bulk-loading people and allocation data
- **PI Management** — Create, activate, complete, clone, and delete Program Increments
- **Data Management** — Export PI data as CSV, load demo data, or reset the database
- **Cost Rates** — Role-based daily cost rates filterable by hub and department
- **Global Search** — Instantly find people, teams, ARTs, and solutions from anywhere
- **Keyboard Shortcuts** — Navigate the app without leaving the keyboard
- **Auto-Updates** — Built-in updater checks for new releases automatically
