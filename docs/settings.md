# Settings

The Settings page is divided into three tabs, each with its own documentation page:

- **[PI Management](pi-management.md)** — Create, activate, complete, clone, and delete Program Increments.
- **[Data Management](data-management.md)** — Export PI data, load demo data, reset the database, and view app info.
- **[Cost Rates](cost-rates.md)** — Role-based daily cost rates filterable by hub and department.
